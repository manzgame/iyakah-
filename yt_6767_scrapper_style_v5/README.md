# 6767 Studio Downloader - Scrapper Style Update v4

Update v4:

- Notifikasi dipindahkan ke bagian atas layar.
- Sistem toast dibuat anti-spam: hanya satu notifikasi aktif yang tampil, notifikasi sama dalam waktu dekat diabaikan.
- Setelah Proses Video atau Proses Musik selesai dan hasil muncul, halaman otomatis scroll ke box hasil.
- Pilihan Video MP4 / Thumbnail diberi jarak lebih rapi dari tombol Proses Video.
- Perbaikan kecil: request API dirapikan dari duplikasi method.

Catatan: resolusi YouTube yang statusnya Link belum ada tetap tidak bisa didownload dari frontend karena provider tidak memberi direct URL.
